# Apex TOC Action

Automate your markdown Table of Contents with ease.

Apex TOC Action is a simple GitHub Action that automatically generates and updates a 'Table of Contents' in your markdown files (e.g., READMEs, documentation) on every push. Keep your project documentation pristine and navigable.

## Features:
- Automatic TOC generation
- Supports READMEs & other markdown files
- Configurable heading levels
- Lightweight & fast
- Seamless GitHub integration
